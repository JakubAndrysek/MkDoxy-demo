## Installation

### Tools
- python 3.6 or newer -> `sudo apt install python3`
- Pip -> `sudo apt install python3-pip`
- Git -> `sudo apt install git`
- Doxygen -> `sudo apt install doxygen`

### PIP Packages
```bash
pip3 install -r requirements.txt
```

## Usage

Serve MkDocs locally:

```bash
mkdocs serve
```

Build MkDocs:

```bash
mkdocs build
```
